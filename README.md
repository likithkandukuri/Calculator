# Advanced Calculator
A simple yet advanced Python calculator built to practice coding, logic development, and input handling.  
This version extends basic arithmetic with percentage, power operations, and a detailed operation history.

## Features
- Handles **addition, subtraction, multiplication, division, percentage, and power**
- Maintains a **step-by-step history** of all operations
- Provides **error handling** for invalid inputs and division by zero
- Allows users to **continue calculations** without restarting the program
- Clean and easy-to-read console interface

## Requirements
- Python 3.8
